# 🔥 My First Java Calculator

Simple command-line calculator built with Java. My first programming project from Shamli, UP!

## ✨ Features
- ✅ Addition (+)
- ✅ Subtraction (-)
- ✅ Multiplication (*)
- ✅ Division (/)
- ✅ Error handling (divide by zero)

## 🚀 How to Run
```bash
1. Download Calculator.java
2. Open cmd/terminal
3. cd Desktop
4. javac Calculator.java
5. java Calculator
```

## 💻 Example Usage
=== Simple Calculator ===
Enter first number: 10
Enter operator (+, -, *, /): +
Enter second number: 5
Result: 15.0
